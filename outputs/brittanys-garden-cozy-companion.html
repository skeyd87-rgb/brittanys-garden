How To 🌱
The quick version for using Brittany's Garden.
1. Add the first plant
Go to Planters, tap Add a plant, then save its name, type, planter, care status, and notes.
2. Use Today for care
Today only shows plants that need water, trimming, or harvesting. Mark care when it is done.
3. Use Care Board for the overview
Care Board groups plants by what needs attention so Brittany can scan the whole garden fast.
4. Edit setup and location
Setup stores the garden name, selected location, temperature unit, and in-app care preferences. Location must be selected from search results.
5. Add to phone home screen
Open the app link on the phone, use the browser share menu, then choose Add to Home Screen.
🔔
About nudges
This version does not send push notifications. It tracks care inside the app. Real phone reminders would need notification permission plus a small backend or scheduled notification system.

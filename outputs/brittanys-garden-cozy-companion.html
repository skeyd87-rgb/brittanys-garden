How To 🌱
The quick version for using Brittany's Garden.
1. Add the first plant
Go to Planters, tap Add a plant, then save its name, type, planter, care status, and optional notes.
2. Use Today for care
Today starts with a Care Plan and an all-plant 7-day calendar, then shows plants that need water, trimming, feeding, or harvesting. Mark care when it is done.
3. Use Care Board for the overview
Care Board groups plants by what needs attention so Brittany can scan the whole garden fast.
4. Edit setup and location
Setup stores the garden name, selected location, temperature unit, and in-app care preferences. Location must be selected from search results. Weather comes from Open-Meteo and needs internet to refresh.
5. Add plant directions
When adding or editing a plant, choose Water every, Feed every, and Check harvest every from the dropdowns. Add a Harvest cue such as "pick when red" so Brittany knows what to look for. Each completed action rolls the next date forward from the day it is logged.
6. Scan seed packages
In Add or Edit Plant, use Seed package to scan a barcode or QR code. Some browsers do not support direct barcode reading yet, so the code can also be typed manually. Use guide fills common care defaults when the plant type is recognized.
7. Use AI carefully
Plant detail pages show built-in guide notes. Private AI advice works only after an AI helper URL is added in Setup. The app does not store or expose an AI API key in the browser.
8. Add to phone home screen
Open the app link on the phone, use the browser share menu, then choose Add to Home Screen.
🔔
About nudges
This version does not send push notifications. It tracks care inside the app. Real phone reminders would need notification permission plus a small backend or scheduled notification system.
